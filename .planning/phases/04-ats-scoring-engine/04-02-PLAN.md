    2. Implement `extractNLPKeywords(text: string, language: 'fr' | 'en'): string[]`:
       - If `language === 'en'`: use compromise NLP
         - `const doc = nlp(text)`
         - Extract noun phrases: `doc.nouns().out('array')` -> lowercased
         - Extract bigrams: `(doc as any).bigrams()` -> map to `.normal` property, filter to 2-word terms

    3. Implement `computeTFIDF(jobTerms: string[], cvText: string, jobText: string): Map<string, number>`:
       - For each term in jobTerms:
         - TF = frequency in jobText / total words in jobText
         - IDF = Math.log((2 + 1) / (inCV + inJob + 1)) + 1 (smoothed, 2-document corpus)
         - inCV = 1 if cvText contains term (word-boundary match), else 0
         - inJob = 1 (always in job text)
       - Return Map<string, number> of term -> tf*idf weight
       - Keep under 25 lines per RESEARCH recommendation

       - If no keywords extracted, return { score: 0, suggestions: ['Job description has no extractable keywords'] }
       - Compute TF-IDF weights: `computeTFIDF(keywords, cvText, jobDescription)`
       - For each keyword, check if present in cvText using word-boundary regex (reuse pattern from computeKeywordMatch)
       - Score = sum of matched keyword weights / sum of all weights * 100, clamped 0-100
       - Generate suggestions for top 5 missing high-weight keywords: "Consider adding keyword: {term}"
