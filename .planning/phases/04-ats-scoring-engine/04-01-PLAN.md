       - Define internal `SubScoreResult = { score: number; suggestions: string[] }`

       - SVG icon check: placeholder, always give full points per D-05
       - Total = sum of all checks, clamp 0-100
       - Generate suggestion strings for each failing check

    4. Implement `scoreContent(cvData: CVData, language: 'fr' | 'en'): SubScoreResult`:
       - Collect all bullet points from `cvData.experience[].description`
       - If no bullets, return { score: 0, suggestions: ['Add experience bullet points'] }
       - Metrics check: % of bullets matching `METRICS_REGEX` (/\d+%|\$[\d,.]+|€[\d,.]+|\d{2,}/) -- target >= 50%, proportional score out of 25 pts
       - Weak verb check: % of bullets NOT starting with weak verb (from WEAK_VERBS[language].weak) -- proportional score out of 25 pts
       - Bullet length check: % of bullets with 5-30 word count -- proportional score out of 20 pts
       - Essential sections check: count [experience, education, skills, summary] present in includedSections or inferred from data -- out of 15 pts. Use cvData arrays: experience.length > 0, education.length > 0, skills.length > 0, personal_info.summary exists
       - Skills not empty: skills array has items -- 15 pts if yes, 0 if no
       - Total = sum, clamp 0-100
       - Generate suggestions for failing checks
