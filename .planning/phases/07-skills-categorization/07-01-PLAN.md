<objective>
Add automatic skill categorization at LinkedIn import, bilingual category display in all templates, and ATS-compatible plain text rendering for skills sections.

Purpose: LinkedIn-imported CVs currently dump all skills into a single "Competences cles" bucket. ATS parsers and recruiters expect skills grouped by type (Technical, Tools, Methodologies, Soft Skills). This plan adds a client-side dictionary for instant categorization, bilingual category names, and ATS-mode plain text output.

Output: Working skill categorization pipeline from import through rendering, with full test coverage.
</objective>
